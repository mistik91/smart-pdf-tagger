# Security

Smart PDF Tagger handles local PDF files and project JSON that may contain sensitive document data. Please avoid sharing private PDFs, project files, credentials, or exploit details in public issues.

## Reporting

For security concerns, use GitHub private vulnerability reporting if it is available for the repository. If it is not available, contact the maintainer privately before publishing details.

Please include:

- Affected version or commit.
- Reproduction steps.
- Impact and expected behavior.
- Any relevant logs or screenshots with sensitive data removed.

## Supported Version

Security fixes target the latest public release.

## Local Data

Project JSON files embed PDF content as base64. Treat project files with the same care as the original documents.
